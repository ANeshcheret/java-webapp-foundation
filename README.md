### Error Case:  If “Unknown version of tomcat was specified” is shown on this window,
- Make sure if tomcat is unzipped in specified directory.
- Check whether unzip has failed.

- Eclipse is checking following files in the tomcat directory.
```
conf/catalina.policy
conf/server.xml
conf/web.xml
conf/context.xml
conf/tomcat-users.xml
conf/catalina.policy
conf/catalina.properties
lib/catalina.jar
```

![img](https://riversun.github.io/img/jweb/addsv03.png
 "img")


### Error Case:  If you erase tomcat from “Servers”, nothing is displayed here,and you may not be able to select Tomcat.

- In that case,try following steps.
```
1.Close Eclipse
2.Open [workspace]/.metadata/.plugins/org.eclipse.core.runtime/.settings
3.delete files below.
org.eclipse.wst.server.core.prefs
org.eclipse.jst.server.tomcat.core.prefs
4.Restart Eclipse
```

 "img")
